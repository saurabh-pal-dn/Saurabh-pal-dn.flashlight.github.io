🌐 Install Flashlight

Step 1: Show your bookmarks bar

Ctrl + Shift + B or ⌘ + Shift + B

Step 2: Drag this button to your bookmarks

🔦 Flashlight Mode

Step 3: Click the bookmark on any site to activate

⚠️ Heads Up

  • This is a harmless piece of front-end JavaScript. It collects zero data.
  • Some highly secure websites (like banks or GitHub itself) use strict Content Security Policies (CSP) that block external scripts from running. It won't work on those sites.
  • Works entirely locally in your browser.
  • Clicking the bookmark a second time toggles the flashlight off.
